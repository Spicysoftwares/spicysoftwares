Lua Script System

Power your game with embedded scripting

The Script System allows you to integrate Lua scripts directly into your game entities. Write gameplay logic, handle input, play animations, and trigger audio without recompiling your engine.

Key Features

  • Lua scripting integration with full API support
  • Real-time entity control and transformation updates
  • Input handling (keyboard detection for W, A, S, D, Space, E, Q)
  • Animation playback control via script commands
  • Audio playback system with async sound support
  • Hot-reload scripts during runtime
  • Custom print function for debugging

Available Lua Functions

  • print(message) - Output debug messages to console
  • IsKeyDown(key) - Check if a key is currently pressed
  • PlayAnim(animName) - Trigger an animation by name
  • PlayAudio(filename) - Play a sound file asynchronously
-- Example Lua Script (player_movement.lua)

function OnUpdate(dt, x, y, z)
    local speed = 5.0
    
    if IsKeyDown("W") then
        z = z - speed * dt
    end
    
    if IsKeyDown("S") then
        z = z + speed * dt
    end
    
    if IsKeyDown("A") then
        x = x - speed * dt
    end
    
    if IsKeyDown("D") then
        x = x + speed * dt
    end
    
    if IsKeyDown("SPACE") then
        PlayAnim("Jump")
        PlayAudio("jump_sound.wav")
    end
    
    return x, y, z
end

The Script System runs during every frame update, allowing smooth and responsive gameplay. Each entity with a ScriptComponent can have its own unique behavior defined in Lua. The system is included in RenderCore and ready to use out of the box.
